# 爆款公众号文章写作

## 核心能力

帮助用户写出适合微信公众号平台传播的高质量文章正文。专注于：

1. **文章结构设计**：选择合适的爆款结构模型
2. **内容撰写**：撰写吸引人的开头、有说服力的正文、有力的结尾
3. **风格优化**：去除 AI 痕迹，注入真实人味
4. **传播优化**：提升文章的转发、评论、收藏率

## 工作流程

### 第一步：理解需求

收集以下关键信息：

- **主题/选题**：文章要表达什么核心内容
- **目标受众**：写给谁看（职场人/学生/宝妈/创业者等）
- **文章类型**：干货教程/观点论述/故事分享/清单盘点/热点评论
- **预期长度**：短文（800-1200字）/中等（1500-2000字）/长文（2500字以上）
- **传播目标**：希望读者做什么（转发/评论/收藏/关注）

### 第二步：选择结构模型

根据文章类型选择合适的结构模型（详见 references/article-templates.md）：

| 文章类型 | 推荐结构 | 特点 |
|---------|---------|------|
| 干货教程 | 痛点-方案-行动 | 实用性强，易收藏 |
| 观点论述 | 颠覆-论证-重塑 | 反常识，易转发 |
| 故事分享 | 故事-冲突-启示 | 情感强，易共鸣 |
| 清单盘点 | 价值承诺-清单-建议 | 信息密度高，易收藏 |
| 热点评论 | 事件-深挖-趋势 | 时效性强，易讨论 |
| 情感共鸣 | 情感钩子-经历-升华 | 打动人心，易转发 |

### 第三步：规划文章结构

使用以下框架规划文章：

```
开头（150-200字）：
- 钩子：痛点/场景/悬念/数据
- 目标：3秒内抓住注意力

过渡（100-150字）：
- 放大问题或引入话题
- 让读者产生"这说的就是我"的共鸣

主体（800-1500字）：
- 核心内容展开
- 每300-500字一个小标题
- 用案例、数据、对比增强说服力

结尾（100-150字）：
- 总结升华或行动召唤
- 金句收尾，引发传播
```

### 第四步：撰写与优化

#### 开头撰写技巧

使用以下5种技巧之一：

1. **痛点直击**："你是不是也经常……"
2. **场景代入**："凌晨2点，……"
3. **数据冲击**："数据显示，90%的人……"
4. **悬念设置**："为什么越努力越穷？"
5. **权威引用**："巴菲特说……"

#### 正文撰写原则

- **短段落**：每段不超过3-4行（手机端）
- **口语化**：用"你"直接对话，避免书面语
- **案例具体**：用具体人名、数字、场景
- **节奏变化**：长短句结合，制造呼吸感
- **金句点缀**：每800-1000字一个金句

#### 结尾撰写类型

1. **总结升华**：回顾核心，提炼洞察
2. **行动召唤**：明确告诉读者做什么
3. **情感共鸣**：温暖收尾，引发转发
4. **悬念留白**：引发思考，促进评论
5. **互动提问**：直接提问，增加互动

#### 去除 AI 痕迹

撰写完成后，检查并去除 AI 写作痕迹（参考 humanizer-zh skill）：

- 删除填充短语："值得注意的是"、"必须要说的是"
- 打破公式结构：避免三段式列举
- 变化节奏：混合句子长度
- 注入个性：加入个人观点和真实感受
- 具体细节：用具体代替抽象

### 第五步：质量检查

发布前对照检查清单：

- [ ] 开头3秒能否抓住注意力？
- [ ] 每段都有价值，没有废话？
- [ ] 段落是否够短（手机端不超过4行）？
- [ ] 是否用了"你"来对话读者？
- [ ] 是否有至少一个金句？
- [ ] 结尾是否有行动引导？
- [ ] 读起来是否口语化、自然？
- [ ] 是否避免了 AI 痕迹？

## 使用示例

### 示例1：撰写干货教程

**用户请求**："帮我写一篇关于时间管理的公众号文章，目标受众是职场新人"

**处理流程**：
1. 确定类型：干货教程类 → 使用"痛点-方案-行动"结构
2. 规划结构：
   - 开头：描述职场新人时间管理的痛点
   - 主体：3个具体方法（番茄工作法、时间块、精力管理）
   - 结尾：行动召唤
3. 撰写内容：使用口语化表达，加入具体案例
4. 检查优化：去除 AI 痕迹，添加金句

### 示例2：优化已有文章

**用户请求**："帮我优化这篇文章，让它更适合公众号传播"

**处理流程**：
1. 分析原文：结构、语言风格、内容密度
2. 识别问题：段落过长、缺少案例、结尾无力等
3. 提供优化：
   - 重写开头，增加钩子
   - 拆分长段落，增加小标题
   - 补充具体案例和数据
   - 强化结尾，添加金句
4. 解释修改：说明每处修改的原因

### 示例3：学习写作技巧

**用户请求**："公众号文章开头怎么写才能吸引人？"

**处理流程**：
1. 加载 references/writing-framework.md
2. 讲解开头的5大技巧
3. 提供具体示例
4. 让用户尝试，给予反馈

## 最佳实践

### DO（这样做）

1. **先理解再写作**：充分理解用户需求和目标受众
2. **结构先行**：先规划结构，再填充内容
3. **短段落**：每段2-3行，制造呼吸感
4. **具体案例**：用真实案例代替抽象概念
5. **口语对话**：用"你"直接对话读者
6. **金句点缀**：每800-1000字一个易传播的金句
7. **行动导向**：结尾明确告诉读者下一步做什么

### DON'T（避免这样做）

1. **不要长段落**：超过5行的段落让读者窒息
2. **不要书面语**：避免"值得注意的是"、"综上所述"等
3. **不要抽象空洞**：不说"很多人"，说"小张"；不说"很多数据"，说"调研1000人"
4. **不要平铺直叙**：要有情绪起伏，制造对比
5. **不要完美主义**：先完成再完美，允许不完美
6. **不要自嗨**：时刻想着读者能获得什么价值

## 注意事项

1. **真实性优先**：不要为了爆款而编造，真实才有力量
2. **价值导向**：每篇文章都要给读者带来价值（知识/情感/行动）
3. **避免标题党**：内容要配得上标题，否则伤害信任
4. **持续迭代**：爆款公式在不断变化，保持学习和调整
5. **尊重原创**：参考案例仅供学习，不要抄袭
